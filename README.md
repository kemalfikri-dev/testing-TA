## Fitur
- Autentikasi (Login & Register)
- Role Admin & User
- CRUD Categories
- CRUD Menus + Search
- Soft Delete & Hard Delete
- CRUD Orders
- Update Status Order
- JOIN 4 Tabel (Users, Orders, Order Details, Menus)
